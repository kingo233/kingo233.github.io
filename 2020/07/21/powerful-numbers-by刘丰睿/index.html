powerful numbers by刘丰睿

发表于 2020-07-21 | | 阅读次数:

Volodya always has a passion for math and numbers. And it seems to him that a integer number is powerful only when it is positive and divisible by each of its nonzero digits. He is eagerly to count the quantity of the powerful numbers in the given interval a,b. Can you help him?

Input Specification:

The first line contains one integer n (1≤n≤10) which is the number of queries. Then n lines follow, each line contains two integers a and b (1≤a≤b≤$9∗10^{18}$)

Output Specification:

Your answer should contains t numbers - answers to the queries, one number per line which is the number of powerful numbers for each given interval [a, b].
