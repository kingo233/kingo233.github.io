发表于 2020-07-19 | | 阅读次数:

Given a positive integer n, you need to find out the minimum integer k such that for any subset T of the set {1,2,…,n} of size k, there exist two different integers that u is a factor of v.

Input Specification:

The first line contains an integer T (1≤T≤105) indicating the number of test cases.

Each of the following T lines contains an integer n (2≤n≤109) describing a test case.

Output Specification:

For each test case, output a line containing an integer which indicates the answer.
